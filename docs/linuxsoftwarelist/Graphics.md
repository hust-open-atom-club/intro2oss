## 图像浏览

| 软件名称 | 简介 | 官网 | 安装方式 | 卸载方式 |
| -------- | ---- | ---- | ---- | -------- |
| Eye of GNOME(EOG) | Gnome桌面默认图片查看器 | <https://wiki.gnome.org/Apps/EyeOfGnome> | `sudo apt-get install eog` | `sudo apt-get purge eog` |
