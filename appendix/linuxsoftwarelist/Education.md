# 教育科学

## 教育软件

### Texmaker

- 介绍

Free cross-platform LaTeX editor(Windows, MacOsX, Linux)

- 下载位置

[下载位置](https://www.xm1math.net/texmaker/download.html)

- 安装方法
 
```
sudo apt-get install texmaker
```

- 卸载方法

```
sudo apt-get purge texmaker
```
